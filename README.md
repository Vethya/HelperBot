# Telegram Helper Bot
A python3 modular helper bot for telegram.
## Installation
### Termux
1. Install Python
- `pkg install python`
2. Clone this repository
- `git clone https://github.com/Vethya/HelperBot`
3. Change directory
- `cd HelperBot`
4. Install requirements
- `pip3 install -r requirements.txt`
## Run the bot
### Termux
- `python3 -m helper`
